Правила
Правила пользования ботом

1. Получение вайфу

  • Вайфу можно получать командой /im не чаще одного раза в 2 часа.
  • Полученные вайфу автоматически попадают в твой гарем (/storage).
  • Каждая вайфу приносит очки в зависимости от редкости.

2. Секретные ключи

  • Активируются командой /promo КОД.
  • Каждый ключ можно активировать только один раз.
  • Ключи могут содержать награды: очки, сброс кулдауна, вайфу или донат.

3. Донат-магазин

  • Донаты сокращают время ожидания между получениями вайфу.
  • Оплата производится через YooMoney.
  • После оплаты нажми «Проверить оплату» для активации.

4. Запреты

  • Запрещено использование ботов, скрипов и любых автоматизированных средств.
  • Запрещён брутфорс промокодов.
  • Запрещено создание множественных аккаунтов для обхода ограничений.
  • За нарушения предусмотрены ограничения доступа по усмотрению администрации.

5. Групповые чаты

  • Бот в групповых чатах отвечает только на команды (начинаются с /) и ответы на свои сообщения.
  • Обычные текстовые сообщения в группе ботом игнорируются.

6. Ответственность

  • Администрация не несёт ответственности за контент вайфу.
  • Пользователь самостоятельно несёт ответственность за использование полученного контента.
